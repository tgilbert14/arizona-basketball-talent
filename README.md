# big-12 talent pathways

This Shiny application analyzes how far Big 12 football and basketball recruits 
travel from their high schools to the universities they ultimately attend.

The “Distance Traveled Over Time” creates a visualization that calculates the 
distance traveled for each recruit and creates an interactive scatter plot 
through the years selected (2016-2026) for the selected university. Users can 
change schools, explore individual recruits, compare classes, and view summary 
statistics (mean, median, trend, outliers) that reactively update. 
Data is from 247Sports as of January 2026.
